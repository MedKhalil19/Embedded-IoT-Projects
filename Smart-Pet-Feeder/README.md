# 🐾 Smart Pet Feeder

## 📌 Project Overview

The **Smart Pet Feeder** is an Internet of Things (IoT) system designed to automate pet feeding while allowing users to monitor and control the device remotely through a mobile application.

Developed during my engineering internship, this project combines embedded systems, electronics, PCB design, mechanical design, and mobile application development to create a complete connected product.

The goal was to design an intelligent feeding solution capable of providing scheduled and manual feeding while offering a simple and intuitive user experience.

---

# 🎯 Project Objectives

- Automate pet feeding
- Enable remote monitoring and control
- Develop a reliable embedded system
- Design a custom electronic circuit
- Create a dedicated PCB
- Design a mechanical enclosure
- Develop a user-friendly mobile interface
- Integrate hardware and software into a complete IoT solution
